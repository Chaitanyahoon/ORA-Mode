## Installation

1. Clone this repository.
2. Open `BizarreStand.csproj` in Visual Studio 2022.
3. Keep the **"Visual Studio extension development"** workload installed.
4. Build the Solution (**Ctrl+Shift+B**).
5. Run the generated `.vsix` file in `bin/Debug/net472/` OR press **F5** to debug.
